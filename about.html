# わたしたちの取り組み

わたしたちは，国立大学法人香川大学創造工学部開設科目である｢ロジカル思考演習｣の担当教員チーム(自称:スーパーロジカルチーム)です． ここでは，わたしたちの取り組みについて紹介します． (未完成な部分もあり，随時更新中です)

# どのようなチーム?

国立大学法人香川大学には，創造工学部という工学系の学部があります． こちらの学部が全コース向けに開設している科目の一つ｢ロジカル思考演習｣の担当教員で構成されたチームです．

# ロジカル思考演習とは?

ロジカル思考演習は大学入学初頭に受講できる科目で，ロジカル思考について学べることはもちろんのこと， 大学での積極的な学びの姿勢を身につけることを目標としています． 受講生は330人程度で，8クラスを5名の教員が担当しています．

# 何に着目し，何に注力している?

近年、学習指導要領に記載され，推進が求められている教育体制として｢主体的対話的で深い学び｣があります． しかし，この目的の達成のためになされるアクティブラーニングは各学校組織に委ねられるところも多く，前衛的な取り組みをする高校もあれば，一部の授業での実現に終わる高校も存在します．

そのため、大学ではアクティブラーニングに慣れ親しむ層とそうでない層とが混在しており， 後者にとっては演習授業での心理的安全性に不安を抱える状況が危惧されています．

ロジカル思考演習は大学初年次に開講される演習授業であり、大学での学びのスタートアップとなる重要な授業に位置づけられていました．

そこで，いかに積極的な学びの姿勢を実現できるのか，担当教員間で頭をひねり，検討を重ねた末に積極的な学びの絶えない演習授業の実現にこぎつけました．

# これからどうしたい?

わたしたちは，大学生が学びの場で積極的な姿勢をみせることに不安を感じない，心理的安全性の高い演習授業の設計について考え，実践を繰り返しています． ロジカル思考演習を積極的に受講した学生には，起業や社会貢献活動，研究･制作活動で成果を出した方も多くいます．

今後はわたしたちの取り組みに関心のある方々と連携して課題を共有し，大学人の使命として積極的な学びの姿勢に向けたノウハウを発信し続けたいと考えています．
